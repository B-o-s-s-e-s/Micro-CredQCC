What exactly is a favicon?

Favicon is an icon in front of the tabs of each website.

How are favicons created?

Favicon is creating a logo, image, and icon using the numbers in Binary and both numbers and letters in Hexadecimal.

How did I create the favicon for this site?

  1. Open Pixelation Link
  2. Chose image width: 11, Image height: 11, Bits per pixel : 3, Binary
  3. Used graph paper to draw out the image and decide the color
  4. Typed the number
